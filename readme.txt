== Installation ==

Upload the Geetest plugin to your blog, Activate it, then enter your [Geetest.com API key](http://geetest.com/).

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setting your Geetest.com API key
